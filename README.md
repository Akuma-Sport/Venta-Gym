# Venta-Gimnasio

Bienvenido al repositorio del proyecto **Venta-Gimnasio** de Akuma Sport. Este proyecto tiene como objetivo crear un sitio web para un gimnasio que ofrezca información sobre servicios, membresías y contacto.

## Descripción del Proyecto

**Venta-Gimnasio** es un sitio web estático diseñado para proporcionar información sobre los servicios ofrecidos por Akuma Sport. Actualmente, el proyecto se encuentra en una etapa inicial y se están implementando las funcionalidades básicas.

## Tecnologías Utilizadas

- HTML: Se utiliza para la estructura del sitio web.
- CSS: Para el estilo y diseño (se puede agregar más adelante).
- JavaScript: Para funcionalidades interactivas (se puede agregar más adelante).

## Instalación

Para clonar el repositorio y comenzar a trabajar en el proyecto, usa el siguiente comando:

```bash
git clone https://github.com/Akuma-Sport/Venta-Gym.git
```

## Uso

1. Abre el archivo `index.html` en tu navegador para ver la página principal.
2. Realiza cambios en el código según sea necesario.
